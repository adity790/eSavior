# ♻️ eSavior

**eSavior** is an AI-driven framework for sustainable **E-Waste Management** that uses Machine Learning to predict a device's **remaining lifespan**, **disposal readiness**, **current market value**, and **depreciated value**. The project helps users make informed decisions regarding repair, reuse, resale, or responsible disposal of electronic devices.

## 🚀 Features

- Remaining Lifespan Prediction
- Disposal Readiness Classification
- Current Price Estimation
- Depreciated Price Prediction
- Automated Data Preprocessing
- Random Forest-based Machine Learning Models
- Model Serialization using Joblib

## 🛠️ Tech Stack

- Python
- Pandas
- Scikit-learn
- NumPy
- Joblib
